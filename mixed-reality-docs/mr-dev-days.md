## <a name="about-altspace"></a>Altspace について

[AltspaceVR](https://altvr.com/) は、世界中の友好的な人々と共にライブ ショーやミートアップ、楽しいクラスなどに参加できる素晴らしい場です。 すべては仮想現実のすばらしい技術のおかげです。  Altspace は、2017 年に Microsoft ファミリの一員となりました。このたびは仮想空間でイベントを開催する方法をご紹介できる機会を持てたことをとても光栄に思っています。

Altspace では、次のプラットフォームをサポートしています。HTC Vive、Oculus Rift、Go、Quest、Samsung GearVR、Windows Mixed Reality、PC での 2D モード。  Mac ユーザーは、Parallels または Bootcamp を使用して、Windows を使って AltspaceVR にアクセスする必要があります。

### <a name="creating-an-account"></a>アカウントの作成

アカウントを作成するには、次の 2 つの方法があります。

- 最も効率的なのは、セットアップに選択した AltspaceVR クライアントをダウンロードし、インストールを完了してから、起動する方法です。 プログラムを開始すると、アカウントを作成するためのオプションが表示されます。  システムの最小仕様については、[こちら](https://help.altvr.com/hc/en-us/articles/115003538414-Minimum-System-Specifications)を参照してください。

- もう 1 つのオプションは、[オンライン アカウント サインアップ](https://account.altvr.com/users/sign_up)にアクセスする方法です。

### <a name="troubleshooting"></a>トラブルシューティング

Altspace で問題が発生した場合は、[トラブルシューティングのページ](https://help.altvr.com/hc/en-us/sections/115000792634-Troubleshooting)をご確認ください。 それでも問題が解決しない場合は、[サポート リクエスト](https://help.altvr.com/hc/en-us/requests/new)を提出してください。

### <a name="additional-tips"></a>その他のヒント

- AltspaceVR を初めて使用する場合は、最初のイベントの前に、時間をかけてチュートリアルを完了してください。
- (必要に応じて) その他のヒントやコツを得るには、AltspaceVR 101 のいずれかのイベントに参加することをお勧めします。
- 後からクライアントで簡単に検索できるように、イベント前に、興味のあるものをお気に入りとして登録します。
- 他の仮想ミーティングとは異なり、AltspaceVR はイマーシブ環境であるため、他の参加者のビューを遮ってしまう可能性があります。 全員が快適に操作できるように、他の参加者のスペースとビューを尊重してください。
- イベントで話したいすべての相手を友達登録することで、友達を AltspaceVR ホームに招待してプライベートで会話できます。
- マイクの近くに立つと、自分の声がイベントに参加している全員にブロードキャストされます。 このため、話すときには場所に注意してください。

## <a name="frequently-asked-questions"></a>よく寄せられる質問
このセクションは、皆様から寄せられるご質問が増えるに応じて頻繁に更新されます。

**Q** - VR ヘッドセットがなくても参加できますか?  
**A** - もちろんです。 このイベントでは VR ヘッドセットを使用した場合に最適なエクスペリエンスが得られますが、この形で参加できない人にも開かれたものにしたいと考えています。  Altspace には、イベントに参加するために使用できる PC クライアントがあります。 "***Altspace には現在、Mac プラットフォームで動作するクライアントはありません (Parallels または Bootcamp を使用して、MAC で AltspaceVR を使うための Windows インストールを作成することをお勧めします)。***"  
  
**Q** - どうすれば 2D エクスペリエンスに参加できますか?  
**A** - Windows Store またはストリームを介して、PC から [Altspace クライアントをダウンロードする](https://altvr.com/get-altspacevr/)オプションが用意されています。 クライアントをダウンロードしたら、アカウントを作成してログインする必要があります。 2D エクスペリエンス内で移動する方法については、[こちら](https://help.altvr.com/hc/en-us/articles/115003528373-How-do-I-control-my-avatar-with-Mouse-Keyboard-)を参照してください。
  
**Q** - どのような人が Mixed Reality Dev Days に参加するのですか?  
**A** - このイベントは、複合現実の領域で仕事をしたりプレイしたりするすべての人を対象としています。 ほとんどが開発者向けのコンテンツになりますが、開発者以外のさまざまな職務の人々にとって興味深いコンテンツも含まれています。 必ずスケジュールの最新情報をチェックして、提供予定のコンテンツの種類などをご確認ください。  

**Q** - AltspaceVR でイベントが見つかりません  
**A** - PC のブラウザーのリンクに従い、AltspaceVR の Web サイトにログインし、イベントの [Interested] をクリックします。  これにより、AltspaceVR クライアントのイベント ビューの [Interested] タブにイベントが表示されます。  また、[Featured] 一覧にもイベントが表示されます。  

**Q** - イベントに参加しましたが、ビューを遮る多数のチュートリアル メッセージがあります  
**A** - 初回のチュートリアルを最初に完了せずにイベントに参加した場合、ビューを遮らないようにするには、これを完了する必要があります。  イベント前に AltspaceVR にログオンし、チュートリアルを完了してこのイマーシブ環境を使用するための基礎を学習することをお勧めしているのは、このためです。
