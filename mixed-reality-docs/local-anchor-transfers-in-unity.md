### <a name="importing"></a><span data-ttu-id="b6b21-140">込ん</span><span class="sxs-lookup"><span data-stu-id="b6b21-140">Importing</span></span>

<span data-ttu-id="b6b21-141">送信元からバイトをすべて受信した後、データを*WorldAnchorTransferBatch*にインポートして、ルートゲームオブジェクトを同じ物理的な場所にロックすることができます。</span><span class="sxs-lookup"><span data-stu-id="b6b21-141">After receiving all of the bytes from the sender, we can import the data back into a *WorldAnchorTransferBatch* and lock our root game object into the same physical location.</span></span> <span data-ttu-id="b6b21-142">注: インポートに失敗する場合があり、再試行が必要になることがあります。</span><span class="sxs-lookup"><span data-stu-id="b6b21-142">Note: import will sometimes transiently fail and needs to be retried:</span></span>

```
// This byte array should have been updated over the network from TransferDataToClient
private byte[] importedData;
private int retryCount = 3;

private void ImportRootGameObject()
{
    WorldAnchorTransferBatch.ImportAsync(importedData, OnImportComplete);
}

private void OnImportComplete(SerializationCompletionReason completionReason, WorldAnchorTransferBatch deserializedTransferBatch)
{
    if (completionReason != SerializationCompletionReason.Succeeded)
    {
        Debug.Log("Failed to import: " + completionReason.ToString());
        if (retryCount > 0)
        {
            retryCount--;
            WorldAnchorTransferBatch.ImportAsync(importedData, OnImportComplete);
        }
        return;
    }

    this.gameRootAnchor = deserializedTransferBatch.LockObject("gameRoot", this.rootGameObject);
}
```

<span data-ttu-id="b6b21-143">*WorldAnchor* *オブジェクト*は、 *lockobject*呼び出しによってロックされた後に、世界の同じ物理的な位置に保持されますが、Unity 座標空間内の他のユーザーとは異なる場所にある可能性があります。</span><span class="sxs-lookup"><span data-stu-id="b6b21-143">After a *GameObject* is locked via the *LockObject* call, it will have a *WorldAnchor* which will keep it in the same physical position in the world, but it may be at a different location in the Unity coordinate space than other users.</span></span>
